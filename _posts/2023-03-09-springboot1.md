# 术语
## RESTful
* RESTful，一种通用的前后台交互方式
* RESTful一般是指这种格式：
  * 使用HTTP POST(或GET)进行数据交互
  * 请求数据和应答数据均为JSON格式（或XML）

## POJO
* pojo（Plain Ordinary Java Object）：普通的Java对象，其实就是简单的JavaBean实体类。对应数据库里的某一张表，pojo里的每一个属性都和该表中的字段一一对应。
